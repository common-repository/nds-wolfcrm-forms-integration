Plugin que permite enviar a WolfCRM los datos obtenidos a través de formularios de Ninja Forms de forma automática.
https://www.wolfcrm.es/

== Description ==

WolfCRM & Ninja Forms Integration trabaja conjuntamente con Ninja Forms para que puedas crear formularios compatibles con Contactos, Clientes y/o Clientes Potenciales de WolfCRM.
Podrás programar que, cuando el usuario rellene el formulario web, los datos se añadan de forma automática a WolfCRM para gestionarlos de forma directa.

== Installation ==

Para instalar el plugin, sigue los siguientes pasos:

1. Es necesario tener activado el API de WolfCRM y disponer de claves.
2. Es necesario tener instalado y activado Ninja Forms.
2. Instala WolfCRM & Ninja Forms Integration
3. En el panel de administración de WordPress, en la opción WolfCRM Forms, rellena los datos de parametrización que has obtenido de WolfCRM
4. Crea tu formulario en Ninja Forms, estableciendo en cada campo el atributo ID (Clave del campo) como el campo del servicio web de WolfCRM en el que quieres que vuelque la información.
5. Añade al formulario un ACTION HOOK, según la entidad a la que quieras insertar.
